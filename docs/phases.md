# AI Career Copilot - Phases

## Phase 1 — Core SaaS Foundation
- Authentication (Email Login, Google Login, Forgot Password, Email Verification, JWT Sessions, Protected Routes)
- Dashboard (Resume Score, Job Applications, Interview Score, ATS Score, Weekly Progress, AI Usage, Recent Activities)
- User Profile (Personal Details, Skills, Experience, Education, Resume History, Preferences)

## Phase 2 — Resume Center
- Resume Upload (Drag & Drop PDF, DOCX Support, Version History)
- AI Resume Analysis (ATS Score, Grammar Review, Formatting Suggestions, Missing Keywords, Technical Skills, Soft Skills, Weak Points)
- Resume Builder (Create resume from profile, Templates, Download PDF)
- Resume Comparison (Compare V1 vs V2, Show improvement)

## Phase 3 — Job Match Center
- Paste Job Description
- AI gives: Match Score, Missing Skills, Matching Skills, Suggested Projects, Resume Improvements

## Phase 4 — Interview Center (Flagship Module)
- AI Mock Interview (Choose domain: Frontend, Backend, MERN, Java, Python, AI, DevOps)
- AI Interview (AI asks Technical Questions -> User Answers -> AI Evaluates -> Next Question)
- HR Interview (Behavioral Questions, STAR Method, Evaluation)
- Coding Interview (Embedded Monaco Editor, AI checks Correctness, Time Complexity, Space Complexity, Code Quality)
- Interview Report (Overall Score, Strengths, Weaknesses, Roadmap)

## Phase 5 — Skill Gap Analyzer
- Compare Resume and Job Description
- Generate Missing Skills -> Learning Roadmap -> Courses -> Projects

## Phase 6 — AI Learning Center
- Example Goal: Become Backend Developer
- AI creates 30-Day Plan -> Daily Tasks -> Progress Tracking

## Phase 7 — Job Tracker
- Instead of Excel. Kanban Board.
- Track: Applied -> OA -> Interview -> Rejected -> Offer

## Phase 8 — Analytics
- Charts: Applications, Interview Scores, Resume Score, Weekly Progress, Skills Growth, AI Usage

## Phase 9 — Portfolio Analyzer (Unique)
- Paste GitHub Profile
- AI analyzes: Repository Quality, README Quality, Languages, Activity, Suggestions

## Phase 10 — Cover Letter Generator
- Generate Custom Cover Letter for each Job Description

## Phase 11 — Notifications
- Email reminders for Interviews, Applications, Roadmap

## Phase 12 — Admin Panel
- Users, Analytics, Reports, Usage, Feedback

## UI Pages
Landing Page, Login, Signup, Dashboard, Resume Center, Resume Analysis, Resume Builder, Job Matcher, Interview, Interview History, Skill Gap, Learning Roadmap, Job Tracker, Analytics, Profile, Settings, Pricing (Optional), Admin
