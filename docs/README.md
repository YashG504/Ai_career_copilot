# AI Career Copilot

An AI-powered career platform that helps students and professionals optimize resumes, prepare for interviews, track job applications, identify skill gaps, and accelerate their job search.

## Tech Stack
### Frontend
- Next.js 15
- React
- Tailwind CSS
- TypeScript
- Shadcn UI
- Framer Motion

### Backend
- Node.js
- Express.js
- TypeScript

### Database
- MongoDB

### Authentication
- Clerk (for fast/secure auth with Next.js) or Custom JWT + Google Login
- Google Login

### AI
- Gemini / OpenAI
- LangChain (later if needed)

### Deployment
- Vercel (Frontend)
- Railway/Render (Backend)
- MongoDB Atlas

## System Modules
- Authentication
- Dashboard
- Resume Center
- Job Match Center
- Interview Center
- AI Learning Center
- Job Tracker
- Analytics
- Profile
- Admin

## What makes this project different?
Most student projects stop at: Resume upload -> AI response -> Done.
We are building a real product with user accounts, persistent data, dashboards, analytics, reusable AI services, clean architecture, polished UI, and deployment.
